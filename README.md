# Homelab Project Portfolio

## Overview

This repository documents hands-on IT infrastructure, networking, storage, backup, and Docker projects built in a self-hosted homelab environment.

These projects focus on real-world problem solving, system configuration, and troubleshooting across multiple technologies.

---

## Projects

### 1. Secure Remote Access & Network Configuration

* OpenVPN configuration (TCP 443 for restricted networks)
* Cloudflare Tunnel setup with custom domain (no port forwarding)
* SSH remote access configuration
* DNS configuration and troubleshooting (Cloudflare, router settings)
* Network diagnostics and log analysis

👉 [View Project](./projects/secure-remote-access-networking/README.md)

---

### 2. Storage & Backup Automation System

* Multi-drive storage setup (NVMe, SSD, HDD)
* Automated backups using rsync and cron jobs
* File system management and mount configuration
* SMART monitoring and drive health checks
* Troubleshooting permissions and data integrity issues

👉 [View Project](./projects/storage-backup-automation/README.md)

---

### 3. Docker Homelab Environment

* Ubuntu-based Docker host with Portainer
* Containerized media stack (Jellyfin, qBittorrent, *arr services)
* Container networking and volume management
* VPN integration for container traffic (Gluetun)
* Troubleshooting container deployment and connectivity issues

👉 [View Project](./projects/docker-homelab-environment/README.md)

## Skills Demonstrated

This portfolio is intended to demonstrate practical, hands-on experience in IT systems, infrastructure, and troubleshooting for entry-level IT, support, and cybersecurity roles.
